## Текущее состояние проекта (на 2026-03-29)

### Структура проекта
```
alex-dispenser-project/
├── arduino/dispenser_controller/  - Прошивка для ESP32
├── packages/
│   ├── frontend/                  - React UI (Vite)
│   ├── backend/                   - Express.js API
│   ├── shared/                    - Общие типы TypeScript
│   └── rpi-service/               - Сервис для Raspberry Pi
├── deploy/                        - Скрипты деплоя
├── database.sql                   - Схема БД
├── README.txt                     - Документация
└── SETUP.txt                      - Инструкция по настройке
```

### Технологии
- Frontend: React 19, Vite, TypeScript
- Backend: Express.js, TypeScript
- Hardware: ESP32 (Arduino), Raspberry Pi
- Serial: Протокол общения ESP32 <-> RPi

### Основные функции
- NFC аутентификация пользователей
- Управление наливанием воды через реле
- Веб-интерфейс для мониторинга и сервисного режима
- Серийный протокол команд (SET_RED, START_FILL_500ML, и т.д.)

### Известные зависимости
- react ^19.0.0
- express ^7.1.0
- ws (WebSocket)
- EEPROM (ESP32)

### TypeScript статуc
- Ошибок компиляции нет
